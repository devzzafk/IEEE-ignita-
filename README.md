IEEE Ignite — Event Website 

A modern, responsive single-page event website developed for IEEE Ignite, a flagship technical event organized by the IEEE Student Branch, LBSITW.

TECH STACK :
html
css
javascript (vanilla js )

FEATURES IMPLEMENTED :

Home section with event branding, date, venue, and event overview
About IEEE section with IEEE and Student Branch information
9 categorized event cards across Technical, Professional, and Interactive tracks
Organized event schedule timeline
Interactive FAQ section with expandable/collapsible answers
Smooth navigation between all sections
Event filtering by category
Real-time event search functionality
Fully responsive design for desktop, tablet, and mobile devices

ADDITIONAL ENHANCEMENTS MADE :

Unique PCB-inspired visual identity tailored for an engineering event
Custom design system built with CSS variables
Sticky navigation with active section highlighting
Animated circuit-trace hero section with traveling spark effect
Scroll-reveal animations for enhanced user engagement
Mobile-friendly navigation menu
